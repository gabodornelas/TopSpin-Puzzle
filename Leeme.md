# TopSpin Solver
## Cómo compilar y ejecutar
#### Para compilar:
* Se debe tener instalado el paquete makefiley el compilador gcc, luego se ejecuta el comando _make_ en la terminal dentro de la ubicacion del proyecto.
* Para Linux existe un paquete que contiene ambos ejecuta el siguiente comando:
```bash
sudo apt update
sudo apt install build-essential
#### Para ejecutar:
* Ejecutar en la terminal dentro de la ubicacion del proyecto el siguiente comando:
```C
./topspin n1 n2 n3 n4 n5 n6 n7 n8 n9 n10 n11 n12 salida.txt
